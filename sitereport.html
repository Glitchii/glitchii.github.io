Files & file structure

There are 6 HTML files in this assignment project. All HTML files are in the root directory of the project. I made seperate folders for other files, such as js directory for JavaScript files, CSS for CSS files, imgs for images etc. All the above folders are in the assets directory. The index.html file is homepage. Most pages are linked in the header including the contact page and home page & some pages won't be displayed based on the active page. Since this is actually my personal website, I decided not to include all pages eg. sitereports or blog in the the menu since they will be useless to most people coming to view my website.

While every HTML file has its own CSS and (if used) JS files, I also have global files such as index.css and index.js. These files content required and used for all pages. index.css is imported in all CSS files with CSS's `@import` rule. You can find this line on the first line of all CSS files except the index.css and 404.css. index.js is imported through all HTML files. index.css has styles for content like the header which is on all pages and index.js has things like event listeners (etc) for the header and other global elements.
